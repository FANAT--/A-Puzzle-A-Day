# A-Puzzle-A-Day
Решает головоломку A-Puzzle-A-Day, перебирая всевозможные варианты. Код оптимизирован для быстрого поиска решений.

## Запуск
> A-Puzzle-A-Day-Solver.exe DD.MM

где DD.MM - день в году
